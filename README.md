# WealthFlow

A production-ready personal budgeting app with Firebase Auth, Firestore, PWA support, and an iOS-inspired glass interface.

## Features
- Sign in / sign up / password reset
- Expense and income tracking
- Budgeting and goal tracking
- Currency conversion via a free exchange-rate API
- Offline-friendly PWA shell

## Development
```bash
npm install
npm run dev
```

## Deploy to Firebase Hosting
```bash
npm run build
firebase login
firebase deploy
```
